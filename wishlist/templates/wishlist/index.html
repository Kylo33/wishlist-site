Hello! I made this website for people to create and share wishlists in an efficient manner. The idea is simple: add links to items from any website (or multiple!) and share it with your friends and family!

Features

This website has a few features that set it apart.

  • No ads, no tracking, no affiliate links!
  • Users can follow wishlists, which sends them a single email (max one per day) if changes were made.
  • With notes, you can convey any extra information about each product!
